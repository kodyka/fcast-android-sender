## Listener lifetime

In this step, `CaptureResultBus.setListener(captureListener)` lives in
`onCreate`. Step 03 adds:

```java
@Override
protected void onDestroy() {
    CaptureResultBus.setListener(null);
    super.onDestroy();
}
```

âŠpaired with all the other unregister calls (display listener, GL thread,
etc.). Until step 03 lands, the listener leaks if the activity is destroyed
without `onCreate` running again. This is acceptable as a short-lived
intermediate state â the LBM register on `main` already has the same leak.
