# 02.7 — Testing

| Test                                                                | How                                                                                              | Pre-merge required? |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------|---------------------|
| Debug APK builds                                                    | `./gradlew --no-daemon :app:assembleDebug`                                                       | Yes                 |
| Capture flow works (UI → consent → casting starts)                  | Manual on a real device.                                                                          | Yes                 |
| Repeated grant works                                                | Manual: start capture, stop, start again. No `IllegalStateException`.                            | Yes                 |
| Rotation does not break capture                                     | Manual: start capture, rotate device. Capture continues.                                          | Yes                 |
| `LocalBroadcastManager` is gone                                     | `rg -n 'LocalBroadcastManager' app/src` returns empty.                                            | Yes                 |
| Dependency is gone                                                  | `./gradlew :app:dependencies \| grep -i localbroadcast` returns empty.                            | Yes                 |
| No leaked listener                                                  | JVM test (added in step 10) calls `CaptureResultBus.requireNoListener()` between scenarios.       | If step 10 already landed |
| Slint headless tests pass                                            | `cargo test -p fcastsender --test ui_snapshots`                                                  | Yes                 |
| Android Lint                                                         | `./gradlew :app:lint`                                                                            | Yes                 |

## Pre-merge command block

```bash
rg -n 'LocalBroadcastManager' app/src
./gradlew --no-daemon :app:assembleDebug
./gradlew :app:lint
./gradlew :app:dependencies --configuration debugRuntimeClasspath | grep -i localbroadcast
cargo test -p fcastsender --test ui_snapshots
```

The first command must return zero lines; the fourth likewise.

## Listener-leak detection

JVM test sketch (lives in step 10's full source):

```java
@After
public void noListenerLeak() {
    CaptureResultBus.requireNoListener();
}
```

If any individual test forgets to call `CaptureResultBus.setListener(null)`,
the `@After` hook fails with a clear "listener leaked: …" message.
