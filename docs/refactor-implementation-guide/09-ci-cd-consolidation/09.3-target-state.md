## Workflow responsibility table

| Workflow                       | Trigger                                              | Outputs                                  |
|--------------------------------|------------------------------------------------------|------------------------------------------|
| `android-debug-apk.yml`        | `push`, `pull_request`                                | Debug APK artifact (30 days).             |
| `android-release-apk.yml`      | `release` (published) or `workflow_dispatch`         | Signed release APK attached to release.  |
| `gstpop-smoke.yml`             | `pull_request` with `backend/gstpop/**` changes       | Pass/fail.                                |
| `slint-viewer-smoke.yml`       | `pull_request` with `ui/**` changes                   | Pass/fail.                                |
| `ui-lint.yml`                  | `pull_request` always                                  | Pass/fail.                                |
| `symbol-stability.yml`         | `pull_request` always                                  | Pass/fail with diff if changed.            |
