## Why the project is behind

The project depends on:

- **GStreamer Android.** GStreamer's Android prebuilts have their own
  release cadence; 1.28.0 has been stable since 2024.
- **Slint Android backend.** Slint has its own AGP compatibility list.
- **JNI symbol stability.** Bumping anything risks NDK toolchain
  changes.

Each upgrade carries triple compatibility risk; conservative pinning is
the safe default.

## Dependent steps

- Step 08 (Kotlin migration) **requires** Kotlin plugin (sub-PR 11.6).
- Step 09 (CI / CD) and step 10 (tests) **assume** Gradle 9 + AGP 8.x or 9.x.
- Step 12 (performance pass) **requires** baseline profile support (AGP
  8.10+) and ProGuard rules tuning (AGP 9.x improvements).

The build-stack upgrade is a prerequisite for the second half of the
plan. Land 11.1–11.4 before 12 / 13.
