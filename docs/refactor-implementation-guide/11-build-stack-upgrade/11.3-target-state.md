## Why these particular versions?

- **AGP 9.0.x:** Newest stable major. AGP 9.2.0 (if available at the
  time of the upgrade) is acceptable; the recipe targets 9.0.x because
  9.2 changed manifest-merger defaults in ways that need a separate
  follow-up PR.
- **Kotlin 2.3.x:** Newest stable. Required for `data object` semantics
  in step 08 and for forward-compatible coroutines.
- **NDK r28c:** AGP 9 default. Older NDKs work but generate deprecation
  warnings.
- **compileSdk/targetSdk 36:** Required to publish to Play Store from
  August 2026 onwards (Android 16 enforcement). Earlier targets work
  today but won't pass Play review next year.
- **Gradle 9.5.0:** Latest 9.x. Substantially faster configuration
  caching.

## Skip target

If the team chooses not to adopt AGP 9 or NDK r28c (e.g., GStreamer
incompatibility), the *minimum* upgrade is:

- Gradle 9.5
- AGP 8.13
- compileSdk 35, targetSdk 34
- NDK r25c (unchanged)
- Kotlin 2.3

This is a safer intermediate target. PRs 11.1–11.3 + 11.6 cover it.
PRs 11.4, 11.5, 11.7 can be deferred until GStreamer 1.30 lands.
