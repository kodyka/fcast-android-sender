
## Manual smoke script

```bash
# 1. Install
./gradlew installDebug

# 2. Trigger a null sticky restart
adb shell am startservice --user 0 -n org.fcast.android.sender/.ScreenCaptureService

# 3. Capture logcat
adb logcat -d ScreenCaptureService:V '*:S' | head -10

# Expected output contains:
#   Ignoring sticky/null restart or unexpected action=null
```
