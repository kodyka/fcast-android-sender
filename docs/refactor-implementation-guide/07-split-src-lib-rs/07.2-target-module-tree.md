## Per-module responsibility table

| Module                | Owns                                                                   |
|-----------------------|------------------------------------------------------------------------|
| `app`                 | `App` composition root, `OnceCell<App>` singleton (step 05).            |
| `application`         | `Application` struct, Slint MainWindow wiring, presets/quick-actions.    |
| `backend`             | `MediaBackend` trait, `BackendRegistry`, concrete backend impls.         |
| `command`             | HTTP probes, HTTP graph commands, legacy/test runners.                   |
| `jni_bridge`          | Every Java‚ÜîRust shim, organised by Java callsite.                        |
| `platform`            | `PlatformApp` glue, `PanelStack`, GStreamer init.                        |
| `secret`              | `SecretStore` trait + JNI impl (step 06).                                |
| `android_main`        | `android_main` body ‚Äî composition + UI bootstrap on Android.            |


`jni_bridge` does not depend on `application` directly ‚Äî it goes through
