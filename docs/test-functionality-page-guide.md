# Test Functionality Page — Implementation Guide

Step-by-step guide for creating `ui/pages/test_functionality_page.slint`,
a unified test screen that combines a **camera phone video source**,
**SRT source**, and **image overlay** section with start/stop controls.

> **Guide only — no code modifications.**
> All Slint snippets follow the existing theme-token and `@tr()`
> conventions enforced by `.pre-commit-config.yaml` (no raw hex
> colours, no hard-coded font sizes, use `PanelBridge.push/pop`).

## Step 3 — Create the TestFunctionality state global

**File:** `ui/state/test_functionality.slint` (new)

This is an optional, feature-scoped global that mirrors the Bridge
properties for cleaner encapsulation — following the same pattern used
by `ui/state/mixer.slint`, `ui/state/camera.slint`, etc.

```slint
// ui/state/test_functionality.slint

import { MixerState, SrtSource } from "../bridge.slint";

export global TestFunctionality {
    // ── Camera source (Rust → Slint for readback, Slint → Rust for edits) ─
    in-out property <int>   camera-idx:          1;
    in-out property <int>   resolution-idx:      2;
    in-out property <int>   framerate-idx:       1;
    in-out property <bool>  camera-mirror:       false;
    in-out property <bool>  camera-stabilization: true;
    in-out property <float> camera-zoom:         1.0;

    // ── SRT source ────────────────────────────────────────────────────────
    in-out property <SrtSource> srt-source: {
        slot-id:    "test-srt",
        enabled:    true,
        uri:        "",
        latency-ms: 2000,
        stream-id:  "",
        mix-alpha:  1.0,
        mix-zorder: 0,
        mix-volume: 1.0,
        state:      MixerState.idle,
        last-error: "",
    };

    // ── Image overlay ─────────────────────────────────────────────────────
    in-out property <string> overlay-image-path: "";
    in-out property <bool>   overlay-enabled:    false;
    in-out property <float>  overlay-x:          0;
    in-out property <float>  overlay-y:          0;
    in-out property <float>  overlay-width:      320;
    in-out property <float>  overlay-height:     180;
    in-out property <float>  overlay-alpha:      1.0;
    in-out property <int>    overlay-z-order:    10;

    // ── Test lifecycle ────────────────────────────────────────────────────
    in property <MixerState> state: MixerState.idle;
    in property <string>     error-text: "";

    callback start-test();
    callback stop-test();
    callback pick-overlay-image();
}
```

## Step 4 — Create the page component

**File:** `ui/pages/test_functionality_page.slint` (new)

This is the main deliverable.  The page follows the exact structure of
`camera_page.slint` (PanelHeader + ScrollView + SettingsSections) and
reuses the `SrtSourceRow`-style layout from `mixer_page.slint`.

### Design decisions

| Decision | Rationale |
|----------|-----------|
| `TestStateChip` duplicated from `mixer_page.slint` | Avoids cross-page coupling; lift to `components/` later if reuse grows |
| `ImageOverlayCard` uses `LineEdit` + `Bridge.pick-test-overlay-image()` | Slint has no native file/photo picker; Rust opens the Android intent or desktop dialog and writes back the path |
| Overlay preview is a placeholder `Rectangle` | Real preview requires a `NativeImage` texture pushed from Rust GStreamer pipeline |
| `Bridge.test-*` properties used directly | Simplest approach; swap for `TestFunctionality.*` global if you created Step 3 |

## Appendix: Mapping from Moblin Swift → Slint

| Moblin (SwiftUI) | Slint equivalent | Used in |
|------------------|------------------|---------|
| `Section { }` | `SettingsSection { title: ... }` | All sections |
| `Toggle(isOn:)` | `SettingsToggleRow { toggled(...) }` | Camera mirror, stabilization, overlay enable |
| `Slider(value:, in:)` | `SettingsSliderRow { value <=> ...; changed(...) }` | Zoom, latency, position, alpha |
| `TextEditNavigationView` | `SettingsValueRow { clicked => cycle(...) }` | Camera, resolution, framerate selectors |
| `TextField` / `TextEditor` | `LineEdit { text <=> ...; edited(...) }` | SRT URL, stream ID, overlay path |
| `PhotosPicker(selection:)` | `LineEdit` + `PrimaryButton { clicked => pick-image() }` | Image overlay selection |
| `Image(uiImage:)` | Placeholder `Rectangle` (real preview needs `NativeImage`) | Image preview |
| `NavigationLink { Form { } }` | `PanelBridge.push(Panel.test-functionality)` + full page | Page navigation |
| `List { ForEach { } }` | `for item in model:` repetition | Future dynamic lists |
